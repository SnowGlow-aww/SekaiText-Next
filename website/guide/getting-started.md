# 安装与首次启动

## 下载安装

前往 [下载页](../download) 获取对应平台的安装包：

- **macOS**：打开 `.dmg`，把应用拖入「应用程序」。首次打开请 **右键 → 打开**；如提示「已损坏」，执行 `sudo xattr -cr "/Applications/SekaiText Next.app"` 后重试。
- **Windows**：运行 `.exe` 安装程序。SmartScreen 提示时点「更多信息 → 仍要运行」。

## 首次启动

启动后应用会自动完成初始化，并在后台**拉取剧情目录元数据**（走 CDN 加速，一般数秒完成；此后每次启动自动刷新，保证活动 / 卡面索引始终最新）。主界面左侧为功能导航，中间是剧情索引 / 编辑区，首次进入会有新手导览逐步介绍。

推荐首次启动时做三件事：

1. **浏览设置页**（侧栏齿轮图标），按习惯调整外观与保存目录；
2. 到 **插件市场** 安装需要的插件（自动轴机 + 压制、Live2D 播放器等按需装）；
3. 如安装了 Live2D 播放器，先完成 **依赖资源同步**（见 [Live2D 剧情播放器](./live2d)）。

## 常用设置一览

| 分组 | 代表设置 |
|------|------|
| 外观 | 主题模式（跟随系统 / 浅色 / 深色）、主题色（推しカラー）、界面缩放、字体导入、背景壁纸（蒙版 / 模糊可调） |
| 文件保存 | **译文保存根目录**（自动建档与保存的落盘位置，默认 `~/文稿/SekaiText`）、JSON 下载目录、保存 `\N` 换行符 |
| 编辑 | 字号、索引排序、撤销深度、切换模式时保留剧情 |
| 下载 | 语音源、下载渠道（国内 CDN / GitHub 直连，互为兜底） |
| 快捷键 | 打开 / 保存 / 搜索 / 撤销重做 / 导入校对稿等全部可自定义 |
| 调试 | 调试日志页开关，反馈问题时打开 |

## 自动更新

应用启动时自动检查新版本并横幅提示，下载走国内 CDN（可在设置切换 GitHub 直连）；更新失败时到 [下载页](../download) 手动安装最新版覆盖即可，数据不会丢失。
