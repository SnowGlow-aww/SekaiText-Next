# 安装与首次启动

## 下载安装

前往 [下载页](../download) 获取对应平台的安装包：

- **macOS**：打开 `.dmg`，把应用拖入「应用程序」。首次打开请 **右键 → 打开**；如提示「已损坏」，执行 `sudo xattr -cr "/Applications/SekaiText Next.app"` 后重试。
- **Windows**：运行 `.exe` 安装程序。SmartScreen 提示时点「更多信息 → 仍要运行」。

## 首次启动

启动后应用会自动完成初始化，无需配置端口或后端服务。主界面左侧为功能导航，中间是剧情索引 / 编辑区。

推荐首次启动时做两件事：

1. **浏览设置页**（侧栏齿轮图标），按习惯调整；
2. 如需使用 Live2D 播放器，先在其中完成 **依赖资源同步**（见 [Live2D 剧情播放器](./live2d)）。

## 设置项一览

| 设置 | 说明 |
|------|------|
| 字号 | 编辑器文本显示大小（10–48px） |
| 索引排序 | 剧情索引下拉列表的排列顺序 |
| 保存 `\N` 换行符 | 翻译文件中保留 `\N` 换行标记 |
| 保存语音文件 | 下载并保存语音文件到本地 |
| SSL 验证 | 某些网络环境下可关闭 SSL 证书验证 |
| 暗色主题 | 亮色 / 暗色切换 |
| 调试日志 | 在编辑器底部显示日志面板，排查问题时打开 |

## 自动更新

应用会自动检查新版本并提示更新，更新失败时可到 [下载页](../download) 手动安装最新版覆盖即可。
