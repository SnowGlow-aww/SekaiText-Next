# 剧情翻译

## 打开剧情

在主界面剧情索引中选择要翻译的内容：

- **活动剧情** — 按活动分组，每期 8 话；
- **卡面剧情** — 按角色卡片分组；
- **主线 / 团队剧情** — 各团主线章节。

选中后原文自动载入编辑器，无需手动提取文本。

## 三种模式

左侧边栏切换工作模式，各模式的文档相互独立：

- **翻译** — 对照原文写译文，从零开始；
- **校对** — 导入他人的翻译稿在其基础上修改，改动处自动与原稿做差异高亮；
- **合意** — 先导入翻译稿、再导入校对稿，两稿逐句对比（黄色为翻译稿、绿色为校对稿），在对比结果上敲定最终译文。

## 编辑界面

编辑器为逐句对照式：每行显示 **说话人、原文、译文输入框**。

- 点击行内 **播放按钮** 可试听该句语音，确认语气断句；
- 说话人名可单独统一翻译（人称表统一见 [术语库](./glossary)）；
- 长句可用 `\N` 手动控制字幕换行（是否写入导出文件见设置「保存 \N 换行符」）。

### 键盘操作 <Badge type="tip" text="5.7.0+" />

- `Esc` 退出编辑状态（保留当前行选中高亮）；
- `↑` / `↓` 在行间移动选中，`Enter` 进入编辑；
- 正在编辑 / 选中的行自动滚动到屏幕居中；
- 撤销 / 重做、打开 / 保存、搜索等快捷键见 设置 → 快捷键（全部可自定义）。

### 工具栏

- **闪回 / 术语 / 同步 / 搜索** 为视图开关：闪回显示回忆片段出处，术语高亮命中词条，同步让双栏滚动联动，搜索支持查找与全部替换（仅替换译文）；
- **说话人** 批量统一说话人翻译；**检查** 全文检查未确认行；**统计** 按角色统计台词与字数；
- **清空** 在顶部导航栏（有二次确认，清空后无法撤销）。

## 术语提示

配置术语库后，原文中命中的术语会在编辑时高亮显示并给出译名，保证多人协作用词一致。详见 [术语库与协作](./glossary)。

## 保存与导出

自 **v5.7.0** 起保存是全自动的：

- **编辑即保存**：每次修改后约 1 秒内自动写入当前文档本体，无需手动操作；
- **自动建档**：全新文档在首次编辑后按选中剧情自动创建规范命名的 txt（`保存根目录/故事类型/索引/【模式】标号 标题.txt`）并绑定为当前文档；卡面剧情命名不带 `-01/-02`，直接以「前篇 / 后篇 / 特殊篇」区分（5.7.6 起）；
- **文档身份跟内容走**（5.7.6 起）：保存的文件名与写入目标在载入剧情那一刻就固定——之后再去拉取 / 下载别的剧情不会影响当前文档；「载入」新剧情则开启新文档会话，不会再把新剧情的译文写进上一个剧情已绑定的文件（此前前后篇连续翻译会互相覆盖串档）；
- **「保存」按钮**：直接写入当前文件，从不询问保存位置——首次落盘也按上述规则自动分级建档；
- **文稿目录按钮**（顶栏）：一键在访达 / 资源管理器中打开保存根目录，直达自己的译文文件；
- **更换保存位置**：设置 → 文稿保存路径 → 更换位置，已生成的文稿会自动整体迁移到新目录（同名文件不覆盖）；
- 完成后可直接进入 [自动打轴](./autotiming) 流程生成字幕轴与成品视频。

::: tip
除文档本体外，应用每 30 秒还会另存一份恢复文件；崩溃或断电后，下次启动会提示恢复。
:::

::: tip
各功能按钮的具体位置以应用内为准；界面持续迭代，文字描述可能略有滞后。
:::
