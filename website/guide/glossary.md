# 术语库与协作

术语库让多人翻译的用词保持一致：角色名、专有名词、称呼方式（人称表）统一管理，编辑时自动提示。

## 本地术语库

- 术语按分类管理，可随时增删改；
- 编辑器中命中术语的原文会高亮并显示对应译名。

## 人称表

人称表以矩阵形式管理角色之间的互相称呼，也可直接选中匹配到对应称呼。

## 团队协作（云端同步）

术语库支持连接协作服务器，供团队多人共用。角色分三级，界面与权限随角色变化（首次进入术语库会按你的角色播放对应的分步导览，被提升权限后自动补播新增功能的教程）：

- **翻译** — 拉取云端术语库（增量合并不覆盖本地修改）、提交新术语提案、跟踪自己提案的审核状态；
- **校对** — 额外可审核成员提案（有待审提案时侧栏「术语库」呼吸灯提醒）；
- **管理员** — 额外可一键**上传至线上术语库**（完全替换线上内容，二次确认）、创建团队账号、提升成员等级。

同步安全网：每次从服务器拉到更新，会自动在本地滚动保留最近 10 份 JSON 备份（`glossary/backups/`），误覆盖可随时回滚。

::: tip
协作服务器的地址与账号由管理员分发，在 **账号中心** 登录。
:::
