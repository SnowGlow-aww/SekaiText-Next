# 术语库与协作

术语库保证全组翻译用词一致：角色名、专有名词、称呼方式（人称表）统一管理，编辑时自动提示。

## 本地术语库

- 支持从 **Excel 表格导入** 术语与人称表；
- 术语按分类管理，可随时增删改；
- 编辑器中命中术语的原文会高亮并显示统一译法。

## 人称表

PJSK 翻译的重头戏——「谁称呼谁用什么」。人称表以矩阵形式管理角色之间的互相称呼，编辑对话时自动按说话人与对象给出正确称呼。

## 团队协作（云端同步）

术语库支持连接协作服务器，供翻译组多人共用：

- **云端同步** — 组员登录后拉取团队最新术语库，增量合并不覆盖本地修改；
- **提案审核** — 组员可以提交新术语提案，由管理员审核后进入正式库；
- 数百人规模的翻译团队实测可用。

::: tip
协作服务器的地址与账号由所在翻译组的管理员分发；个人使用时仅用本地术语库即可，全部功能不受影响。
:::
