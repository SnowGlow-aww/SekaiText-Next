# Live2D 剧情播放器

内置播放器在应用内还原游戏剧情演出：角色 Live2D 模型、表情动作、语音、背景与音效同步播放，翻译时可以随时对照演出确认语境。

## 首次使用：同步依赖资源

Live2D 播放需要一批公共依赖资源（模型、动作等），首次使用前在应用内执行 **依赖资源同步**：

- 下载走国内 CDN 加速，个别 CDN 缺失的资源会自动回退到源站获取；
- 同步支持完整性检查：本地文件缺失或损坏时会自动重新下载；
- 若同步结束提示有失败项，多为个别资源上游缺失，重试一次即可，不影响其他剧情播放。

## 播放剧情

在剧情页打开播放器后：

- **步进播放** — 像游戏内一样逐句推进对话；
- **跳转** — 可直接跳到指定句，方便对照某一句的演出；
- 语音、表情、动作与文本同步展示。

## 导入剧情

除了内置索引中的剧情，也支持导入剧情数据进行播放。

::: tip
个别卡片（如吉祥物卡）本身没有表情差分或配音，播放时无表情 / 无语音属正常现象，并非资源缺失。
:::
