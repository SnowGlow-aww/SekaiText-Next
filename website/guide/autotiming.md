# 自动打轴与压制

SekaiText Next 内置 **SekaiCoreEngine** 打轴内核：识别游戏录屏中的对话画面，把翻译好的每句台词自动对齐到视频时间轴，再经 FFmpeg 一键压制输出成品。

## 准备工作

1. **游戏录屏**：完整录制目标剧情的播放过程（对话auto即可）；
2. **完成翻译**：确保该篇剧情的译文已在编辑器中完成。

## 打轴流程

1. 在打轴功能页选择 **视频文件** 与 **对应剧情**；
2. 开始识别 — 引擎逐帧分析画面上的对话文本，与剧本逐句匹配；
3. 完成后自动导出 `.ass` 字幕文件。
4. 使用aegisub检查匹配结果是否准确。

::: warning 匹配结果为 0 或大量错位？
几乎都是 **视频和剧本对不上**：确认选择的剧情与录屏是同一期活动 / 同一话。注意剧情脚本编号与活动实装编号可能存在偏移，错选就会出现「识别失效」的情况。
:::

## 一键压制

打轴完成后可直接压制：内置 FFmpeg + libass，将字幕烧录进视频输出成品文件。

- 支持硬件加速，实测压制速度可提升4～7倍。
- 字幕样式随导出配置生成，可按团队模板调整。

## 常见问题

- **识别很慢**：识别为逐帧分析，时长与视频长度、机器性能相关，属正常现象；
- **个别行时间不准**：手动微调该行后再导出；
- **导出的 ass 只有样式没有轴**：请更新到最新版本（旧版已知 bug，已修复）。
