# 简介

**SekaiText Next** 是一款为「プロジェクトセカイ カラフルステージ！ feat. 初音ミク」（PJSK）剧情翻译打造的桌面应用。它把原文载入、翻译校对、自动打轴和成片压制接成一条连续工作流，同时保留本地文稿、ASS 与 Aegisub 等既有工具：

- **剧情翻译** — 直接载入活动、卡面和主线剧情，逐句对照原文、译文与语音；
- **自动打轴**（插件）— 从录屏中识别对话画面，生成逐句时间轴，并支持分句微调与 Aegisub 双向同步；
- **成片压制**（插件）— 调用内置 FFmpeg，把字幕、团队样式与视频直接合成为成片；
- **Live2D 剧情播放器**（插件）— 在应用内对照角色动作、表情、语音与背景，翻译时不丢语境；
- **术语库与协作** — 集中管理术语和人称表，支持团队提案、审核与备份；
- **插件市场** — 在应用内按需安装和更新扩展功能。
