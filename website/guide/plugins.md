# 插件市场

SekaiText Next 内置热加载插件系统：功能按需扩展，安装后立即生效，无需重启应用。

## 安装插件

1. 打开应用内 **插件市场**；
2. 浏览可用插件列表（列表与插件包均走国内 CDN 加速）；
3. 点击安装 — 下载完成后自动加载，立刻可用。

## 现有插件

- **自动轴机 + 压制** — 打轴 / 分句微调 / Aegisub 同步 / staff 制作人员行 / 一键压制，见 [自动打轴与压制](./autotiming)；
- **Live2D 播放器** — 应用内还原剧情演出，见 [Live2D 剧情播放器](./live2d)。

## 更新与卸载

- 插件有新版本时市场内会提示，一键更新；更新后若界面没有变化，把插件禁用再启用一次即可；
- 插件会声明所需的**最低主程序版本**：主程序过旧时新插件会拒绝安装（避免「插件是新的、主程序是旧的」导致新功能看着在、实际用不了），先升级主程序再更新插件即可；
- 不需要的插件可随时卸载，禁用插件后其相关设置项也会自动隐藏。

## 插件能做什么

插件与主应用共享同一套界面框架，可以添加新的页面、面板或工具，深度扩展工作流。插件由官方市场统一分发。
