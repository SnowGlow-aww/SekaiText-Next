# 插件市场

SekaiText Next 内置热加载插件系统：功能按需扩展，安装后立即生效，无需重启应用。

## 安装插件

1. 打开应用内 **插件市场**；
2. 浏览可用插件列表（列表与插件包均走国内 CDN 加速）；
3. 点击安装 — 下载完成后自动加载，立刻可用。

## 更新与卸载

- 插件有新版本时市场内会提示，一键更新；
- 不需要的插件可随时卸载。

## 插件能做什么

插件与主应用共享同一套界面框架，可以添加新的页面、面板或工具，深度扩展工作流。插件由官方市场统一分发。

::: tip 开发者
想为 SekaiText Next 编写插件？欢迎到 [GitHub 仓库](https://github.com/SnowGlow-aww/SekaiText-Next) 提 Issue 联系作者获取插件开发说明。
:::
