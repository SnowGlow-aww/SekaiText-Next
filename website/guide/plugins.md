# 插件市场

SekaiText Next 内置热加载插件系统：市场插件按需安装并可立即生效，无需重启应用；从本地文件安装的插件默认保持禁用，需要确认来源后再手动启用。

## 安装插件

1. 打开应用内 **插件市场**；
2. 浏览可用插件列表（列表与插件包均走国内 CDN 加速）；
3. 点击安装或更新；市场条目必须提供匹配的 SHA-256，校验通过后才会安装并热加载。

SHA-256 用于发现下载损坏或包内容与市场条目不一致。校验值与下载地址来自同一份市场索引，因此它是完整性校验，不等同于独立的发布者身份认证。

## 现有插件

- **自动轴机 + 压制** — 打轴 / 分句微调 / Aegisub 同步 / staff 制作人员行 / 一键压制，见 [自动打轴与压制](./autotiming)；
- **Live2D 播放器** — 应用内还原剧情演出，见 [Live2D 剧情播放器](./live2d)。

## 更新与卸载

- 插件有新版本时市场内会提示，需要用户在市场中明确点击更新，不会在后台静默替换插件代码；
- 新版本加载失败时会自动恢复上一个可用版本；恢复也失败时插件会被禁用，避免下次启动继续加载损坏版本；
- 插件会声明所需的**最低主程序版本**：主程序过旧时新插件会拒绝安装（避免「插件是新的、主程序是旧的」导致新功能看着在、实际用不了），先升级主程序再更新插件即可；
- 不需要的插件可随时卸载，禁用插件后其相关设置项也会自动隐藏。

从文件安装时请选择 `.sekplugin`。本地包安装后默认禁用；启用前应用会再次要求确认，因为插件与主程序共享本地文件、数据和网络访问能力。

## 插件能做什么

插件与主应用共享同一套界面框架，可以添加新的页面、面板或工具，深度扩展工作流。市场插件由官方市场分发，也可以从本地 `.sekplugin` 文件安装自行取得的插件。
