# 下载 SekaiText Next

<DownloadButtons style="margin: 32px 0 40px" />

## 系统要求

| 平台 | 要求 | 安装包 |
|------|------|--------|
| macOS | Apple Silicon (M 系列芯片)，macOS 12+ | `.dmg` |
| Windows | x64，Windows 10 及以上 | `.exe` 安装程序 |

## 安装提示

### macOS

打开 `.dmg` 后将 **SekaiText Next** 拖入「应用程序」文件夹。首次打开如提示无法验证开发者：

1. 在「应用程序」中 **右键 → 打开**，再点「打开」；
2. 如提示「已损坏，无法打开」，在终端执行后重新打开：

```bash
sudo xattr -cr "/Applications/SekaiText Next.app"
```

::: tip
以上是未签名应用在 macOS 上的正常放行步骤，并非应用本身损坏。
:::

### Windows

首次运行如出现 SmartScreen 蓝色提示框，点击 **「更多信息」→「仍要运行」** 即可。

## 更新

应用内置自动更新：检测到新版本时会提示下载安装，全程走国内 CDN 加速，无需手动操作。

也可以随时到 [GitHub Releases](https://github.com/SnowGlow-aww/SekaiText-Next/releases) 手动下载任意历史版本。
